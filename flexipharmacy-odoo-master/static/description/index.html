Multi Shop

    - User can create shop and shop wise load product and category.
    - Shop can be assign to user based on user's company.
    - User can only see own data like POS dashboard,order,session based on assigned shop.
    - Switch one shop to another shop based on user's assigned shops.
    - User can print receipt based on selection in POS configuration either company information or shop information.
  • As per below image you can create shops for different stock.


  • As per below image you can set configuration to enable this feature.

  • You can assign shop to users based on the user's company.


  • After allocated the shop user can see the POS dashboard,order,session based on allocated shop.


  • User can define the product and categories which should be load for particular shop in POS.


  • Switch one shop to another shop and load product and categories accordingly.


  • Print receipt with company details or shop details based on configuration.
